# 💬 WhatsApp Clone

A full-featured WhatsApp clone built with Swift for iOS, replicating the core messaging experience with a clean, native UI.

---

## ✨ Features

- 💬 Real-time messaging
- 📞 Calls tab (voice & video)
- 👥 Contacts & chat list
- 🟢 Online/offline status
- 🔔 Push notifications
- 🎨 Native iOS look and feel
